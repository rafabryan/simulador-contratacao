# SkillMatch JS

Projeto desenvolvido em JavaScript puro para simular a compatibilidade entre um candidato Front-End Júnior e vagas fictícias da área de tecnologia.

## Objetivo

Analisar:

- habilidades do candidato;
- requisitos das vagas;
- habilidades faltantes;
- percentual de compatibilidade;
- vaga mais compatível;
- recomendações de estudo.

## Tecnologias Utilizadas

- JavaScript
- Node.js ou Console do Navegador

## Funcionalidades

✅ Cadastro do candidato  
✅ Cadastro de vagas  
✅ Cálculo de compatibilidade  
✅ Classificação das vagas  
✅ Listagem de habilidades faltantes  
✅ Melhor vaga identificada  
✅ Recomendações de estudo  
✅ Uso de métodos de array  
✅ Uso de classes e herança  
✅ Uso de callback  
✅ Uso de closure  
✅ Uso de Promise e async/await

## Como executar

### Opção 1 — Navegador

1. Abra o navegador Google Chrome
2. Pressione F12
3. Vá até a aba Console
4. Cole o conteúdo do arquivo `skillmatch.js`
5. Pressione Enter

### Opção 2 — VS Code

1. Instale o Node.js
2. Abra o projeto no VS Code
3. Execute:

```bash
node skillmatch.js
```
