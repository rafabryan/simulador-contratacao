# 🚀 SkillMatch JS - Triagem de Currículos


## 📋 Sobre o Projeto
O **SkillMatch JS** é uma *engine* (motor de análise) desenvolvida para automatizar a triagem de currículos em vagas de tecnologia. O sistema atua como um simulador, comparando o perfil técnico de um candidato com os requisitos de vagas de Front-End Júnior, otimizando o fluxo de análise e ajudando na tomada de decisões.

## ✨ Funcionalidades
- **Análise de Habilidades:** Verifica e cruza os dados das tecnologias que o candidato domina com as exigências específicas de cada vaga.
- **Cálculo de Compatibilidade (Match):** Gera o percentual exato de aderência do candidato às vagas cadastradas.
- **Mapeamento de Lacunas (Gaps):** Destaca visualmente quais habilidades técnicas estão faltando no currículo.
- **Recomendação Inteligente:** Gera uma indicação automática de estudos com base nas tecnologias faltantes para alcançar 100% de match na vaga mais aderente.
- **Interface Visual Dinâmica:** Dashboard interativo renderizado dinamicamente via manipulação do DOM.

## 🛠️ Tecnologias Utilizadas
Este projeto foi construído consolidando os fundamentos do desenvolvimento web:
* **HTML5:** Estruturação semântica da aplicação.
* **CSS3:** Estilização componentizada com variáveis globais (Custom Properties), Flexbox e UI responsiva.
* **JavaScript (Vanilla):** Lógica da *engine* utilizando métodos de iteração modernos (`filter`, `forEach`, `map`), manipulação de arrays de objetos e renderização dinâmica no DOM.
